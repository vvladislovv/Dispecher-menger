# Диспетчер задач

Диспетчер задач - это мощное приложение, предназначенное для управления и мониторинга процессов в операционной системе. Оно разработано на Python с использованием фреймворка Flet, что позволяет создавать кроссплатформенные пользовательские интерфейсы.

![Скриншот приложения](./Content/screenshot.png)

## Основные функции

- **Мониторинг процессов**:
  - Отображение списка запущенных процессов
  - Информация о потреблении ресурсов каждым процессом
  - Возможность завершения процессов

- **Управление системными ресурсами**:
  - Мониторинг использования CPU и памяти
  - Отслеживание активности диска и сети
  - Графики производительности в реальном времени

- **Системная информация**:
  - Подробные данные об операционной системе
  - Информация о процессоре, материнской плате и памяти
  - Данные о дисках и сетевых подключениях

## Используемые технологии

- **Python 3.12+**: Основной язык разработки
- **Flet**: Фреймворк для создания интерфейсов на основе Flutter
- **psutil**: Библиотека для получения системной информации и управления процессами
- **Docker**: Инструмент для контейнеризации и развертывания приложения

## Архитектура приложения

Приложение имеет модульную структуру:

- **modules/ui/**: Компоненты пользовательского интерфейса
  - **views/**: Различные экраны приложения
  - **app.py**: Основной класс приложения
- **modules/system/**: Модули для работы с системной информацией
  - **process_manager.py**: Управление процессами
  - **performance_monitor.py**: Мониторинг производительности
  - **system_info.py**: Получение системной информации
- **modules/config/**: Конфигурация приложения
  - **settings.py**: Настройки приложения
- **main.py**: Точка входа в приложение

## Установка и запуск

### Запуск с использованием Python

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/vvladislovv/dispecher-menger.git
   cd dispecher-menger
   ```

2. Создайте виртуальное окружение и установите зависимости:
   ```bash
   python -m venv .venv

   # На MacOS/Linux
   source .venv/bin/activate  
   # На Windows: 
   .venv\Scripts\activate

   pip install -r requirements.txt
   ```

3. Запустите приложение:
   ```bash
   python main.py
   ```

### Запуск с использованием Docker

1. Запустите приложение с помощью Docker Compose:
   ```bash
   docker-compose up
   ```

2. Или соберите и запустите контейнер вручную:
   ```bash
   docker build -t dispecher-menger .
   docker run -p 8550:8550 dispecher-menger
   ```
